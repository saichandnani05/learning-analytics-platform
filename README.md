# Learning Analytics Platform

A cloud-based EdTech analytics solution built at Hackwhat to track student performance, automate data processing, and visualize real-time learning insights.

## Features
- End-to-end ETL pipeline using AWS Lambda, S3, MongoDB Atlas.
- RESTful APIs with Flask for quiz evaluation and performance tracking.
- Frontend dashboard built in React.js.
- Dockerized backend for AWS deployment.
- Sample dataset for demo and testing.

## Setup Instructions

### Backend
```
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```
cd frontend
npm install
npm start
```

### AWS Lambda ETL
Upload `etl_lambda.py` to AWS Lambda.
Configure S3 bucket trigger for `sample_quiz_data.csv`.

## Tech Stack
- Flask, Python, Pandas
- MongoDB Atlas
- AWS (Lambda, S3)
- React.js, Axios
- Docker
